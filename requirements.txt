# Your local environment uses CUDA 12.1 (cu121).
# On HPC, check your CUDA version first:
#   nvcc --version   OR   module load cuda
# Then install the matching build:
#
# CUDA 12.1 (same as local):
#   pip install torch==2.5.1+cu121 torchvision==0.20.1+cu121 torchaudio==2.5.1+cu121 \
#       --index-url https://download.pytorch.org/whl/cu121
#
# CUDA 11.8 (common on older HPC nodes):
#   pip install torch==2.5.1+cu118 torchvision==0.20.1+cu118 torchaudio==2.5.1+cu118 \
#       --index-url https://download.pytorch.org/whl/cu118
#
# CPU only (if no GPU available):
#   pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1
#
# NOTE: torch is intentionally excluded from this file because
# the +cu121 local wheel URL will not resolve correctly on HPC.
# Install torch separately using the commands above BEFORE
# running pip install -r requirements.txt

# ============================================================
# HPC SETUP INSTRUCTIONS
# ============================================================
#
# 1. Create and activate a virtual environment:
#       python -m venv physiorag_env
#       source physiorag_env/bin/activate
#
# 2. Install PyTorch with the correct CUDA version (see above)
#
# 3. Install remaining dependencies:
#       pip install -r requirements.txt
#
# 4. Install Ollama (NOT a pip package — needs separate install):
#       curl -fsSL https://ollama.com/install.sh | sh
#    If no root access, check: module avail ollama
#    Then pull the model:
#       ollama pull qwen2.5:7b
#
# 5. If faiss-cpu is too slow on GPU nodes, replace with:
#       pip install faiss-gpu==1.13.2
#
# 6. langchain-classic is a thin compatibility shim —
#    if it fails to install on Linux, try:
#       pip install langchain-classic==1.0.1 --no-deps
